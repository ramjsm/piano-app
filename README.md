# Piano app

Piano application that records and play the songs created.

## Getting Started


### Prerequisites

Run apollo-server

### Installing

```
$ npm install
$ npm start
```

## Built With

* [React](https://reactjs.org/)
* [Pose](https://popmotion.io/pose/) 
* [Sass](https://sass-lang.com/)
* [ApolloGraphQL](https://www.apollographql.com/)
